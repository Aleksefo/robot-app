inline_response_200_1 Up

status (optional)
String OK - Robot access granted, QUEUED - Request queued
Enum:
OK
QUEUED
position (optional)
BigDecimal Position in queue if status = QUEUED
since (optional)
String If status = OK, the time since robot access was granted format: datetime
remaining (optional)
BigDecimal If status = OK, the remaining time in ms until control is again disallowed
